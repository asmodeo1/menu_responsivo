# menu_responsivo
En este ejemplo hemos creado un menú responsivo solo con HTML y CSS:
- en pantallas grandes se usa un menú que muestar sub submeús con hover
- en pantallas pequeñas usamos el típico menú hamburguesa y el truco de usar inputs de tipo checked para ocultar o mostrar el menún entero y los submenús, pues con hover, aunque podemos simularlo con enlaces, tenemos algunos problemas.

Un incoveniente del uso de los inputs es que, si andamos juagndo con el tamaño de pantalla, al volver a mostrar el menú en tamaños pequeños, peude que aparezca ya expandido por algún checked que quedó marcado
